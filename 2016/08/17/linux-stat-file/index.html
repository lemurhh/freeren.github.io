linux命令——stat、file命令

发表于 2016-08-17 | 分类于 操作系统 |

stat命令

使用ls命令虽然能查看文件的大部分信息，但仍然有部分信息无法获得，stat命令能提供文件系统中文件状态的完整摘要，甚至能显示存储该文件的设备的主设备号和次设备号(主设备号提供设备的类型，次设备号确定具体的设备，详见百科)。
