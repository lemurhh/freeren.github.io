2017.3.6

发表于 2017-03-06 | 分类于 随笔

在美国曾发生过这样一个案例：嫌犯在车厢内多次举枪，随意瞄准目标，并最终在下车时枪击了多个目标。按常理，嫌犯多次举枪总会被人发现，但事实上没有，因为，车上的乘客都在低头玩手机，竟然谁都没有注意到有这么一个危险人物。之前一直觉得这事儿不可思议。然而这几天，在地铁上我仔细观察了几乎每个乘客，结果真是让我震惊：几乎每个人都在低头专注于自己的手机，从小学生到七八十的老人，年轻人更是严重，戴着耳机是普遍现象，整个车厢除了地铁飞驰与空气的摩擦声、到站的广播没有几乎没有任何声音，要知道在高峰期，整个车厢全都是人。这种安静实在让人背后发凉，这是互联网的问题还是人的问题？
这几天见了很多沿街乞讨的，一个同学还问我是不是经常遇见这种现象。我确实经常遇到乞丐，但是很少遇到有人理会他们的，我似乎只见过一个小姑娘投了几枚硬币，其他大部分人都是避而远之。乞丐们有时也缠着、跟着人们。昨天我还看到一个乞丐因为乞讨不成，在抱怨（几乎是骂骂咧咧）：现在的人怎么都这样。
现在的安检措施也是到了一种登峰造极的地步，一天平均都要被安检四次，但事实上，都麻木了，习以为常了，进了地铁站自动摘包，想都不用想。但我也发现，很多安检人员都在打马虎眼，甚至安检的屏幕都没人盯着。安检确实是保障了大家的安全，但也确实反应出了人们之间的不信任，那些蓄意制造事故的人也确实不值得信任。文明极大地地发展了，但这种现象该作何解释。
今天在一处路口，几个交警和一个骑电动车的大妈吵起来了（确切地说是大妈和交警吵），具体原因不清楚，但应该是大妈违反了交通规则，交警要处罚她，但她不认账，质疑交警的执法权。交警回了一句话我认为说的很对，执法权是强制的，不是要你承认才行。许多人在吵吵执法公平的问题，但真当法执在自己头上的时候，就不买账了。
今天在玄武湖公园，突然发现地球真的很美，这个世界真是很奇妙，我们这个星球真的是很独特，它是我们在宇宙的归宿，虽然从数量上来说，它只是浩茫宇宙中的一粒尘埃。
刚刚得知我给心爱女孩的熊猫还躺在她的床上，很意外，都有点想哭。
明天去航空烈士纪念馆，去探访抗战时的中国空军。
