孔林

发表于 2016-12-01 | 分类于 旅游 ， 曲阜 |

孔林，本称至圣林，位于山东曲阜城北1.5公里处，是孔子及其家族的墓地。孔子死后，其弟子们把他葬于鲁城北泗水之上，那时还是“墓而不坟”（无高土隆起）。到了秦汉时期，虽将坟高筑，但仍只有少量的墓地和几家守林人,后来随着孔子地位的日益提高，孔林的规模越来越大。1994年12月被列入《世界遗产名录》。

孔府

发表于 2016-11-30 | 分类于 旅游 ， 曲阜 |

衍圣公府俗称“孔府”，位于山东曲阜城内孔庙东侧，是我国现在唯一较完整的明代公爵府。衍圣公为孔子嫡裔子孙世袭爵位，其职责为管理孔子的祀事及孔氏的族务。
孔府现有规模形成于明弘治十六年（1503年）。清光绪十一年（1885年）一场大火把孔府内宅一扫而光，因此留下的明代原物主要是内宅以外的部分建筑物，即大门、仪门、大堂、二堂、三堂、两厢、前上房、内宅门及东路报本堂等，其余均为清代重建或增建。
1961年3月，国务院列为第一批全国重点文物保护单位；1994年12月，联合国教科文组织列其为世界文化遗产；2007年3月，国家旅游局又列为国家AAAAA级旅游景区。

孔庙

发表于 2016-11-30 | 分类于 旅游 ， 曲阜 |

孔庙是我国历代封建王朝祭祀春秋时期思想家、政治家、教育家孔子的庙宇。它是一组具有东方建筑特色、规模宏大、气势雄伟的古代建筑群。曲阜孔庙则是祭祀中孔子的本庙，位于孔子故里山东曲阜城内，又称“阙里至圣庙”，与南京夫子庙、北京孔庙和吉林文庙并称为中国四大文庙。始建于鲁哀公十七年（公元前478年），历代增修扩建，与相邻的孔府、城北的孔林合称“三孔”。
曲阜孔庙被建筑学家梁思成称为世界建筑史上的“孤例”，现为世界文化遗产、中华人民共和国全国重点文物保护单位，与北京故宫、承德避暑山庄并列为中国三大古建筑群。

字节序—大小端

发表于 2016-11-25 | 分类于 操作系统 |

大端存储(big endian)和小端存储(little endian)

大小端的名字来源于《格列夫游记》：小人国内部分裂成Big-endian和Little-endian两派，区别在于一派要求从鸡蛋的大头把鸡蛋打破，另一派要求从鸡蛋的小头把鸡蛋打破。事实上，大小端是一种直译，如果换一种翻译方式可能会更好：”big endian”译为“高尾端”，”little endian”译为“低尾端”。尾端即为数据的末尾，数据的最低位。高尾端意为尾端存储在高地址，低尾端意为尾端存储在低地址。例如，0x1234567两种存储方式如下图：

为什么会有两种方式

有小端方式是由于：计算总是先从数据的低位开始，低位存储在低地址，电路方便，效率高。
而人类显然更喜欢大端方式，毕竟读写方便，所以，除了计算机的内部处理，其他情景几乎都是使用大端存储，例如网络传输、文件存储。

计算机如何处理数据

计算机不知道什么大端小端，它只知道按顺序读取字节，先读第一个字节再读第二个字节。如果是大端存储，先读到的就是低地址处的高位字节。如果是小端存储，先读到的就是低地址处的低位字节。

字节序的处理

就一句话：

只有读取的时候，才必须区分字节序，其他情况都不用考虑。
处理器读取外部数据的时候，必须知道数据的字节序，将其转成正确的值。然后，就正常使用这个值，完全不用再考虑字节序。即使是向外部设备写入数据，也不用考虑字节序，正常写入一个值即可。外部设备会自己处理字节序的问题。
举例来说，处理器读入一个16位整数。如果是大端字节序，就按下面的方式转成值。buf是整个数据块在内存中的起始地址，offset是当前正在读取的位置。

x= buf[offset] * 256 + buf[offset+1];
如果是小端字节序，用下面的公式转成值。 
x = buf[offset+1] * 256 + buf[offset];

bochs安装及使用AT&T制作引导扇区

发表于 2016-11-24 | 分类于 操作系统 |

bochs安装

可以使用包管理安装。

apt-get install vgabios bochs bochs-x bximage
但这种安装方式是没有调试功能的。如需调试功能，需要源代码编译安装。 
tar -zvxf bochs-2.4.6.tar.gz
cd bochs-2.4.6
./configure --enbale-debugger --enable-disasm
make
make install
我使用的是kali，在执行configure时出现错误提示: > pkg-config was not found,or unbale to access the gtk+ 2.0 package.Install the gtk+ development package,or disable the gui debugger,or the wxWidgets display library (whichever is being used) 解决办法是： 
apt-get install libgtk2.0-dev

配置bochs

安装好bochs后需要配置bochs，通过配置文件实现。bochs默认情况下以当前工作目录下的bochsrc文件为配置文件。以下是我的配置文件：

################################################
#configure file for Bochs
################################################

# how much memory the emulated machine will have
megs: 32

# filename of Rom images, 注意下边要根据你安装目录来调整
romimage: file=/usr/local/share/bochs/BIOS-bochs-latest
vgaromimage: file=/usr/local/share/bochs/VGABIOS-lgpl-latest

#what disk images will be used
floppya: 1_44=lemur.img, status=inserted

#choose the boot disk.
boot: floppy

#where do we send log messages?
log: bochsout.txt

#disable the mouse
mouse: enabled=0

# enable key mapping, using US layout as default.
keyboard:keymap=/usr/local/share/bochs/keymaps/x11-pc-de.map

创建镜像文件

bochs提供了bximage这个工具创建镜像，在shell中输入bximage即可打开。这里，我创建了一个名为”lemur.img”的软盘镜像文件。

引导扇区代码

这是使用AT&T汇编写的引导扇区代码，用来调用BIOS在屏幕上输出”hello”。由于没有清屏，输出结果有些杂乱。

#boot.s
.text
.global _start
_start:    # 程序入口
.code16 #16位代码
movw %cs,%ax
movw %ax,%ds
movw %ax,%es
call print
hlt

print:
movw $bootmessage,%ax
movw %ax,%bp
movw $0x5,%cx
movw $0x1301,%ax
movw $0xc,%bx
movb $0,%dl
int $0x10
ret

bootmessage:
.ascii "hello"
.fill 510-(.-_start),1,0  #填充使达到512个字节
.word 0xaa55  #引导扇区结束标志

编译链接

经过编译链接生成boot.bin文件。

as boot.s -o boot.o
ld –oformat binary -N -e _start -Ttext 0x7c00 -o boot.bin boot.o

注:ld可以被配置为支持多于一种的目标文件. binary表示没有程序头和其他信息，仅仅是一些裸数据。如果没有这个选项，将被默认链接为elf格式。-N把text和data节设置为可读写。- Ttext将text节起始地址设置为0x7c00（在jmp和数据引用等重定位链接时会用到这个参考值），所有的引用地址都是在7c00这个地址上加出 来的。-e选项指定程序入口点

写入镜像文件

使用dd命令将boot.bin文件写入lemur.img镜像文件:

dd if=boot.bin of=lemur.img bs=512 count=1 conv=notrunc

注：最后一个选项是说不截断软盘，因为boot.bin比lemur.img要小

运行

将前面写好的bochs配置文件与lemur.img放在一个目录下，在该目录下运行bochs。

大雁塔

发表于 2016-10-24 | 分类于 旅游 ， 西安 |

大雁塔位于唐长安城晋昌坊（今陕西省西安市南）的大慈恩寺内，又名“慈恩寺塔”。唐永徽三年（652年），玄奘为保存由天竺经丝绸之路带回长安的经卷佛像主持修建了大雁塔，最初五层，后加盖至九层，再后层数和高度又有数次变更，最后固定为今天所看到的七层塔身，通高64.517米，底层边长25.5米。
大雁塔作为现存最早、规模最大的唐代四方楼阁式砖塔，是佛塔这种古印度佛寺的建筑形式随佛教传入中原地区，并融入华夏文化的典型物证，是凝聚了中国古代劳动人民智慧结晶的标志性建筑。
1961年3月4日，国务院公布大雁塔为第一批全国重点文物保护单位。2014年6月22日，在卡塔尔多哈召开的联合国教科文组织第38届世界遗产委员会会议上，大雁塔作为中国、哈萨克斯坦和吉尔吉斯斯坦三国联合申遗的“丝绸之路：长安-天山廊道的路网”中的一处遗址点成功列入《世界遗产名录》。

华清宫 骊山

发表于 2016-10-21 | 分类于 旅游 ， 西安 |

到达

今天的计划是先去骊山再去兵马俑，交通还是很方便的，西安火车站那有去骊山和兵马俑的专线旅游车，车上还有介绍情况的导游。大约一个小时，就到了骊山。因为接连几天下雨的缘故，骊山雾气很重。
骊山对中国历史来说是绕不过去的，骊山见证了一次又一次中国历史进程的重大转折：周幽王烽火戏诸侯导致西周灭亡，周王室开始衰落；秦始皇大修豪华宫殿、陵墓，成为导致秦帝国灭亡的重要直接原因；项羽的鸿门宴未能杀掉刘邦，日后，项羽乌江自刎，刘邦建立起中国历史上极其重要的汉帝国；唐玄宗与杨贵妃在此留下了“在天愿作比翼鸟，在地愿为连理枝”的绝世爱情，最终导致了唐王朝的由盛转衰；张学良、杨虎城在此发动西安事变，中国近代历史发生了重大转折，最终抗日战争胜利，新中国建立，中华民族终经过了“三千年未有之大变”的考验。

华清宫景区主要景点

  • 芙蓉湖风景区（望京门、芙蓉榭、曲桥、鹊桥、骊岫飞泉、盟誓台、连理树）

  • 长生殿（玄境长生殿文化数字体验馆）

  • 九龙湖风景区（飞霜殿、九曲回廊、龙石舫、珍宝馆、艺术馆、禹王殿）

  • 梨园遗址博物馆

  • 唐御汤遗址（海棠汤、莲花汤、星辰汤、尚食汤、太子汤）

  • 西安事变旧址五间厅（望湖楼、桐荫轩、荷花阁、白莲榭）

  • 西安事变旧址兵谏亭（兵谏园、西安事变浮雕、摩崖石刻）

  • 昭阳门、三元洞、翠荫亭、晚照亭、举火楼、遇仙桥、石瓮谷、火神庙、芝兰亭

  • 老君殿遗址

  • 老母殿（朝元阁、上善湖、上德苑、七夕桥、明圣宫、鸟语林）

  • 温泉体验区（紫宸苑、紫极苑、迎阳苑、长汤苑、万寿殿、明华殿、紫云阁、圣智堂、宝积池、香凝苑）

  • 餐饮服务区（御膳苑）

    • 华清宫简介

    华清宫是一座位于唐华清宫遗址之上的皇家宫苑，是我国历史上兴建最早的皇家行宫之一，占地面积约11.5平方公里，西距西安30公里。因其亘古不变的温泉清源、烽火戏诸侯的历史典故、唐玄宗与杨贵妃的爱情故事、西安事变的发生地等而成为国家首批AAAAA级景区、全国重点风景名胜区、全国重点文物保护单位、国家森林公园、世界地质公园。
    华清宫景区融人文历史和自然景观于一体，周、秦、汉、唐等历代帝王都曾在此建有离宫别苑，宫殿建筑遍布骊山上下，主要历史遗存有烽火台遗址、老母殿遗址、老君殿遗址、唐御汤遗址、梨园遗址、西安事变旧址五间厅及兵谏亭等。
    骊山自古苍翠欲滴，犹如一匹骏马。“骊山晚照”是著名的“关中八景”之一
